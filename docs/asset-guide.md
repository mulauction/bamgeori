# 에셋 제작·전달 가이드 (사장님용)

"그래픽/사운드를 더 좋게 하려면 뭘 만들어 주면 되나?"에 대한 정리.
**핵심: 파일을 정해진 폴더에 정해진 이름으로 넣기만 하면 게임이 자동으로 씁니다.** (없으면 기본값 사용)

---

## 🎵 1. 소리 (지금 바로 됨)
**폴더:** `public/audio/` · **형식:** `.mp3` · **자세한 목록:** `public/audio/README.md`

| 예시 파일 | 언제 |
| --- | --- |
| `bgm-yabawi.mp3` / `bgm-race.mp3` / `bgm-fight.mp3` | 각 가게 배경음악(루프) |
| `ambience.mp3` | 밤거리 배경(루프) |
| `cheer.mp3` / `start.mp3` / `beep.mp3` | 환호 / 출발신호 / 카운트다운 |

- 배경음악은 **자연스럽게 반복되는 10~30초 루프**, 효과음은 1~2초.
- 무료 소스: [freesound.org](https://freesound.org), [Pixabay Music](https://pixabay.com/music/), [OpenGameArt](https://opengameart.org).
- ⚠️ 반드시 **저작권 없는(CC0)·본인 제작** 음원만. (스토어 출시 대비)

---

## 🖼️ 2. 텍스처 이미지 (지금 바로 됨 — 거리)
**폴더:** `public/textures/` · **형식:** `.png` · **자세한 목록:** `public/textures/README.md`

| 파일 | 적용 | 규격 |
| --- | --- | --- |
| `wall.png` | 건물 벽 | 512×512, **타일링(끊김 없는)** |
| `road.png` | 도로 | 512×512, 타일링 |
| `ground.png` | 인도·광장·골목 | 512×512, 타일링 |

- **2의 거듭제곱 정사각형**(256/512/1024), 벽·바닥은 반드시 **seamless**.
- 무료 소스: [ambientCG](https://ambientcg.com), [Poly Haven](https://polyhaven.com/textures).
- 게임 내부(야바위 좌판, 경기장 트랙 등) 텍스처 슬롯도 요청하면 추가합니다.

---

## 🧱 3. 3D 모델 (진짜 퀄 업 — 요청 시 연결)
지금 캐릭터·개·닭·건물은 **코드로 만든 임시 복셀**입니다. 진짜 로블록스/마인크래프트급은 **실제 3D 모델**로 교체해야 합니다.

- **형식:** `.glb` (권장) 또는 `.gltf`. **폴더(예정):** `public/models/`
- **무료 제작 툴:**
  - **MagicaVoxel**(무료) — 복셀 모델링. `.obj/.gltf`로 내보내기.
  - **Blockbench**(무료, 웹) — 마인크래프트식 모델. `.gltf` 내보내기 지원.
- **무료 에셋 팩(바로 쓸 수 있음):**
  - [Kenney](https://kenney.nl/assets) — City/Character/Vehicle 복셀 팩 (CC0).
  - [Poly Pizza](https://poly.pizza), [Quaternius](https://quaternius.com) — 로우폴리 무료.
- **우선순위(효과 큰 순):** ① 주인공 캐릭터 ② 개·닭 ③ 탈것(차·자전거) ④ 건물 ⑤ 행인.
- 모델 파일 하나라도 주시면 **로더(GLTFLoader)를 붙여서 임시 복셀과 교체**하겠습니다. (한 개로 먼저 검증 후 확장)
- 규격 팁: 1개 모델 **1만 폴리 이하**, 텍스처 포함(embedded), Y-up, 미터 단위.

---

## 🎬 4. 영상 (선택 — 특정 위치)
동영상은 **화면/전광판 같은 평면**에만 유용합니다(캐릭터·거리는 3D 모델이 답).
- **형식:** `.mp4`(H.264). **쓸 곳 예:** 거리 상단 전광판에 광고/뉴스 영상, 인트로 화면.
- 필요하면 `public/video/`에 넣는 슬롯을 만들고 `VideoTexture`로 붙입니다. 어디에 쓸지 알려주세요.

---

## 정리: 지금 당장 효과 큰 것 3가지
1. **배경음악 3개**(`bgm-yabawi/race/fight.mp3`) → 분위기 즉시 상승. **가장 쉬움.**
2. **벽/바닥 텍스처**(`wall/road/ground.png`, seamless) → 거리 질감 상승.
3. **주인공 캐릭터 모델 1개**(`.glb`) → 주면 로더 붙여서 퀄 대폭 상승.

파일 준비되면 폴더에 넣고 알려주세요. 넣기 애매하면 파일만 주셔도 제가 넣고 배포하겠습니다.
