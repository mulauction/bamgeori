# 머니 싱크(돈 쓸 곳) 로드맵

포인트를 버는 이유 = 쓸 곳이 있어야 함. 소비 사다리를 저가→초고가로 길게 깔고 하나씩 구현한다.
**원칙:** 판돈/승률에 영향 주는 유료 요소 금지(공정성). 소비는 전부 **과시(외형)·편의·공간·권력**에만.

### 구현 순서
1. **착용 과시템 5종** (1번) — 지금 구현. 벌면 캐릭터가 눈에 띄게 달라짐.
2. 탈것 추가(2번) → 3) 공간/상점 인수(맵 반영) → 4) 편의·연출 → 5) 육성.
