# 밤거리 승부 — 게임 기획서 v0.1

> 미니게임 승부 · 자산 경제 · 과시 시스템 기반 3D 복셀 게임
> 버전 0.1 (초안) · 2026. 7. 18.
> 본 문서는 `밤거리승부_기획서_v0.1.docx`를 텍스트로 변환한 참조본이다. 수치·배당·확률은 프로토타입 검증 후 조정한다.

## 1. 게임 개요

| 항목 | 내용 |
| --- | --- |
| 장르 | 승부(베팅) 미니게임 모음 + 경제 시뮬레이션 + 소셜 과시 |
| 플랫폼 | 모바일(안드로이드/iOS) 우선, 웹 병행 검토 |
| 그래픽 | 마인크래프트·로블록스 계열 복셀/로우폴리 3D |
| 조작 | 왼손 가상 조이스틱 이동 + 화면 드래그 시점 회전 + 점프 버튼 (3인칭) |
| 핵심 재화 | 포인트(P) — 게임 내에서만 획득·소모, 현금화 절대 불가 |
| 수익 모델 | 현금 결제는 과시(외형) 아이템 전용. 판돈과 완전 분리 |
| 개발 방식 | 1인 개발 + Claude Code. 로직 우선 완성 후 에셋 교체 방식 |
| 출시 전략 | 해외(스팀/글로벌 스토어) 우선 검토, 한국은 등급분류 전략 확정 후 |

### 1.1 한 줄 컨셉
"밤거리를 걸어 다니며 가게마다 다른 승부를 벌이고, 딴 돈으로 남들과 달라지는 게임." 전 재산을 잃으면 새벽 대리운전부터 다시 시작한다.

### 1.2 핵심 감정
- **승부의 긴장** — 판돈을 걸고 결과를 기다리는 순간
- **승리의 폭발** — 등급별 승리 연출(음악·환호·화면 효과)이 주는 도파민
- **과시의 쾌감** — 거리에서 마주치는 타인과의 시각적 차별화(허영심)
- **몰락과 재기** — 파산이 끝이 아니라 서사가 되는 구조

## 2. 핵심 게임 루프
걷는다 → 가게에 들어간다 → 판돈을 걸고 승부한다 → (승리) 포인트 획득·연출 → 더 큰 판 또는 과시템 구매 → (패배) 포인트 상실 → 잔액 소진 시 노동 미니게임으로 재기 → 다시 거리로.

보조 루프: 육성(내 개·닭을 키워 승률에 개입) / 소셜(구경·응원·대결·조직) / 수집(과시템·부동산·권력).

## 3. 승패 요인 설계 — 3층 구조
순수 운으로만 두면 금방 질리고, 실력만으로 두면 진입장벽이 생긴다. 운 위에 "실력의 착각"을 쌓는 3층 구조로 설계한다.

| 층 | 요인 | 설명 | 적용 예 |
| --- | --- | --- | --- |
| 1층 | 순수 운 | 규칙 설명이 필요 없는 즉석 승부. 신규 유저 진입 통로 | 야바위, 홀짝, 사다리, 뽑기류 |
| 2층 | 육성 | 내 소유의 선수(개·닭)를 먹이·훈련으로 성장시켜 승률에 개입. 소유 애착(이케아 효과)으로 승부의 감정 무게 증가 | 개경주(내 개 출전), 닭싸움(내 닭 육성) |
| 3층 | 캐릭터 스탯 | 캐릭터 능력치가 미니게임을 보조. "내가 잘해서 이겼다"는 귀속 심리 형성 | 눈썰미(야바위 셔플이 느리게 보임), 배짱(닭이 응원에 반응), 손재주(실력 게임 보정) |

스탯은 노동·훈련·업적으로만 올릴 수 있게 하여(현금 구매 불가) "돈으로 실력을 살 수 없다"는 공정성 인식을 지킨다.

## 4. 미니게임 라인업 (목표: 수십 종)
- **4.1 운 승부 (즉석, 30초 컷):** 가위바위보, 홀짝, 사다리타기, 제비뽑기, 야바위(컵 3개), 소주뚜껑 숫자, 달고나 뽑기, 인형뽑기, 붕어빵 복불복, 동전 던지기 연승 챌린지.
- **4.2 관전·베팅형 (경주 계열):** 개경주, 경마·경륜·경정, 바퀴벌레 경주, 달팽이 레이스, 닭싸움, 팔씨름 토너먼트. 핵심은 육성 결합 — 내 선수를 출전시키는 판과 남의 판에 거는 판을 구분한다.
- **4.3 실력 미니게임 (일상 소재):** 딱지치기, 구슬치기, 병뚜껑 알까기, 다트, 눈싸움, 초성퀴즈 스피드전, 편의점 가격 맞히기, 노래방 점수 대결.
- **4.4 노동 (재기 루트):** 대리운전(제한시간 주행), 탁송 배달, 붕어빵 굽기(리듬), 전단지 돌리기, 택배 상하차. 파산 시 유일한 수입원.
- **4.5 확장 원칙:** 한국인이 몸으로 아는 놀이를 우선 게임화 / 게임 1종 = 가게 1곳 / 신작은 "신장개업" 이벤트로 거리에 등장.

## 5. 경제 시스템

### 5.1 재화 구조
| 재화 | 획득 | 사용 | 현금 관계 |
| --- | --- | --- | --- |
| 포인트(P) | 승부 승리, 노동, 출석·업적 | 판돈, 육성 비용, 과시템 구매 | 현금으로 직접 구매 불가, 현금화 불가 |
| 캐시(유료) | 현금 결제 | 과시(외형) 아이템 전용 | 판돈으로 전환 절대 불가 — 규제·심의 방어의 핵심 원칙 |

### 5.2 파산과 재기
- 잔액 0이 되면 "빈털터리" 상태 — 외형이 남루해지고 걸음이 처진다(몰락의 가시화).
- 노동 미니게임만 이용 가능. 소액을 모아 다시 승부 세계로 복귀.
- 재기 성공 유저에게 "재기의 훈장" 등 서사 보상 — 파산을 이탈이 아닌 스토리로 전환.

### 5.3 인플레이션 관리
- 배당 구조는 하우스 에지(환수율 95% 내외)로 설계해 포인트 총량을 자연 회수.
- 과시템·육성 비용·조직 상납이 대형 포인트 싱크 역할.
- 수치는 프로토타입 지표(평균 보유액, 파산율, 세션당 판수)를 보고 조정.

## 6. 중독(몰입) 설계 — 심리 장치

### 6.1 승리 연출 등급제
| 등급 | 조건(예) | 연출 |
| --- | --- | --- |
| 일반 승 | 배당 2배 이하 | 짧은 팡파레 + 포인트 카운트업 |
| 빅윈 | 판돈 5~10배 | 화면 정지 → 슬로우모션 → 폭죽 + 관중 함성 + 진동 |
| 잭팟 | 판돈 10배 이상 또는 고액 | 전용 음악 + 거리 전체 전광판 방송("OO님 개경주 500,000P 획득!") |

패배 연출은 짧고 담백하게 — 패배를 오래 보여주면 이탈한다.

### 6.2 접전 연출 원칙
경주·대결의 접전(코 차이)은 재도전 욕구를 자극한다. 단, 결과를 조작해 인위적 니어미스를 만드는 것은 규제 대상. **접전은 실제 확률 결과의 연출로만 사용한다(승부 자체는 사전 확정, 보여주는 방식만 극적으로).**

### 6.3 리텐션 장치
- 매일 바뀌는 이벤트 경주·특별 배당의 날.
- 출석·연속 접속 보상, 시즌제(명예의 전당).
- 한정판 과시템(FOMO).

### 6.4 윤리·안전 가드레일
- 니어미스 조작 금지, 확률 공개, 미성년 이용 등급 준수.
- 일일 판돈 상한 등 과몰입 방지 옵션 검토.

## 7. 과시(허영) 시스템
설계 원칙: 과시는 랭킹판이 아니라 "거리에서의 마주침"에서 작동한다. 모든 과시 요소는 다른 유저의 화면에 보여야 한다.

### 7.1 외형 계층
| 범주 | 예시 | 효과 |
| --- | --- | --- |
| 몸에 걸치는 것 | 금목걸이, 명품 신발(금색 발자국), 오라 이펙트 | 원거리에서도 식별되는 부의 신호 |
| 움직임 | 부자 전용 걸음걸이, 등장 시 스포트라이트, 전용 이모트 | 정적 아이템보다 강한 시선 유도 |
| 소리 | 전용 BGM, NPC 인사("어서 오십시오, 회장님") | 청각 과시 |
| 탈것 | 자전거 → 중고차 → 스포츠카 → 기사 딸린 세단 | 이동 자체의 계급화 |
| 공간 | 건물주 명패, 상점 인수(간판에 내 이름), 광장 동상, 거리명 개명 | 맵에 영구히 남는 흔적 — 최상위 과시 |
| 몰락 | 파산 시 남루한 옷, 처진 걸음 | 부의 가치를 지탱하는 반대편 |

### 7.2 권력 시스템 (장기 목표)
상점 인수 → 구역 보스(조직) → 시장 선거 → 섬 구매·건국·자기 화폐 발행(초장기 로드맵).

## 8. 소셜 시스템
- **1단계 구경:** 관전·응원·야유·훈수. 큰 판에 구경꾼이 모이는 광경 자체가 콘텐츠. 관전자가 많을수록 승리 연출 강화.
- **2단계 대결·조직:** 비동기 기록 대결부터 시작 → 조직(길드), 구역 장악전, 상납 경제.
- **시기심 장치:** 거리 전광판(오늘의 벼락부자/쪽박). 자산 랭킹은 존재하되 핵심 노출은 "거리에서 마주치는 외형".

## 9. 심의·규제 전략
- 사행성 판정의 핵심은 환전 가능성 — 포인트의 현금화·환전 알선 원천 차단.
- 현금(캐시)은 외형 전용, 판돈 전환 불가 — 웹보드 규제 회피.
- 베팅 직접 모사 시 청불 가능성 — 등급 수용 또는 "참가비-우승상금" 프레임 완화 두 안을 프로토타입 후 결정.
- 해외 우선 출시(스팀 Simulated Gambling 태그 등).
- 육성·실력 요소 비중을 심의 자료에서 강조.
