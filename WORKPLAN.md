# 작업 계획 (자율 세션)

위에서부터 순서대로 실행. 작업 1개 = 커밋 1개 이상, 매 커밋 플레이 가능. 상태: ⬜대기 / 🔵진행 / ✅완료 / ⏸️보류.

전 게임 RTP=0.95 (economy 단일 관리), EV 불변, 결과 사전 확정. 한 판 60초·"다시" 1탭.

## A. 공통 기반
- **A1 게임 레지스트리** ✅ — `games/registry.js`에 게임 등록 → 거리 가게·라우팅 자동 생성. 기존 4종 이관. 완료기준: 기존 4종이 레지스트리 경유로 정상 플레이.
- **A2 economy RTP 엔진** ✅ — `RTP`, `payoutFor(prob)`, `crashPoint()`, `houseRoll()` 등 EV불변 유틸. 완료기준: 유틸 Node 테스트로 EV≈0.95 검증.
- **A3 공통 베팅 UI** ✅ — 칩 + 리스크 슬라이더 + 확률·배당 실시간 표시 + 대형 캐시아웃/스톱 버튼. 완료기준: 리스크 조절 시 배당·확률 즉시 갱신.
- **A4 fx 등급제 + 사운드 점검** ✅ — 기존 fx/audio 확인·보강(등급 임계 economy, 사운드 unlock/음소거). 완료기준: 3등급 연출·사운드 동작.
- **A5 거리 확장/구역** ⬜ — 거리 2~3배 + 야시장 골목·뒷골목 2구역, 구역별 색감. (게임 다수 확보 후 배치 여유 위해 A 후반 또는 B 중간 삽입)

## B. 게임 (문서 번안명·표준 수식, RTP 0.95)
즉석 운 4종(간단, 먼저):
- **B1 홀짝** ✅ — 2택, p=0.5, 배당 1.9x.
- **B2 가위바위보** ✅ — vs NPC, 승 1/3(배당1.85)·무 푸시·패.
- **B3 사다리타기** ✅ — 상단 택1→하단 배수(사다리 은닉), 배수 평균=RTP.
- **B4 제비뽑기** ✅ — 제비 N개 혼합 배수(평균=RTP).

카탈로그 12종:
- **B5 림보(심야버스 몇 정거장)** ⬜ — 목표 T 설정, 도달 정거장≥T 승. p=RTP/T, 배당 T.
- **B6 마인즈(복어 손질집)** ⬜ — 5x5 지뢰M, 안전칸 오픈마다 배수↑, 캐시아웃.
- **B7 크래시(새벽 폭주 배달)** ⬜ — 배수 상승·랜덤 버스트, 스톱 캐시아웃 + NPC 피드.
- **B8 플립(동전 연승)** ⬜ — 앞/뒤 연승, 판당 1.9x, 캐시아웃.
- **B9 하이로우(소주뚜껑 높낮이)** ⬜ — 다음 수 높/낮, 확률기반 배당, 연승·캐시아웃.
- **B10 다이스(야시장 눈금 주사위)** ⬜ — 0~99 초과/미만 임계, 리스크 슬라이더.
- **B11 휠(야시장 돌림판)** ⬜ — 배수 세그먼트 스핀, 리스크별 분포(EV동일).
- **B12 플린코(오락실 구슬 낙하기)** ⬜ — 구슬 낙하, 결과칸 사전확정+경로 역산.
- **B13 타워(달동네 계단 오르기)** ⬜ — 행마다 안전칸 택1, 캐시아웃.
- **B14 킨오(밤거리 복권방)** ⬜ — k개 선택·d개 추첨, 적중수 배당표(RTP).
- **B15 케이스(전당포 유실물 상자)** ⬜ — 포인트 전용, 가중 보상표(EV=RTP).
- **B16 치킨로드(닭싸움장 탈출)** ⬜ — 차선 횡단, 칸마다 생존확률·배수↑, 캐시아웃.
