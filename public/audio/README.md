# 사운드 교체 방법 (사장님용)

이 폴더에 `<이름>.mp3` 파일을 넣으면 **코드 합성음 대신 그 파일이 자동으로 재생**됩니다.
파일이 없으면 기존 합성음이 그대로 나옵니다. (커밋 후 push하면 배포에 반영)

| --- | --- | --- |

## 팁
- 배경음악(bgm-*, ambience, crowd, tension)은 **자연스럽게 반복되는(루프) 짧은 mp3**가 좋습니다.
- 효과음은 1~2초 이내 짧게.
- 저작권 없는 음원(예: freesound, 직접 제작)을 쓰세요.
- 파일명은 정확히 위 표대로(소문자, 하이픈).
