# 텍스처(이미지) 교체 방법

이 폴더에 `<이름>.png`를 넣으면 코드로 그린 기본 텍스처 대신 그 이미지가 **거리에 자동 적용**됩니다.
파일 없으면 기존 그대로. (커밋·push하면 배포 반영)

## 지금 교체 가능한 이름
| 파일 이름 | 적용 위치 | 권장 |
| --- | --- | --- |
| `wall.png` | 모든 건물 벽 | **이어붙어도 자연스러운(seamless/타일링) 텍스처** |
| `road.png` | 도로(찻길) | seamless, 가로 방향 반복 |
| `ground.png` | 인도·광장·골목 바닥 | seamless |

## 규격
- **정사각형, 2의 거듭제곱 크기**: 256×256 또는 512×512 권장(1024까지 OK).
- 벽/바닥은 **타일링 되는(끊김 없는) 이미지**여야 반복해도 티가 안 남.
- 무료 소스: [ambientCG](https://ambientcg.com), [Poly Haven](https://polyhaven.com/textures), [Kenney](https://kenney.nl/assets) (전부 상업적 무료).

더 많은 슬롯(간판·캐릭터 등)이 필요하면 말씀해주세요. 슬롯을 추가하겠습니다.
