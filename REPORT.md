- **A2 RTP 엔진** ✅ — payoutFor/crashPoint/winByProb/settleMultiplier. 기존 게임 RTP도 0.95로 정렬(야바위 2.0→2.85, 개경주 재산출). Node EV 검증.
- **A3 공통 베팅 UI** ✅ — 칩 + 리스크 슬라이더 + 확률·배당 실시간 표시 + 대형 캐시아웃 버튼.
- **A4 fx 등급제 + 사운드** ✅ — 일반/빅윈/잭팟(임계 economy) + WebAudio 합성음(기존 구현 확인).

## 아침에 폰으로 확인할 포인트
1. 거리를 걸으면 **가게 20여 곳**이 3구역(밤거리/야시장/뒷골목)으로 늘어서는가. 각 간판으로 입장되는가.
2. 각 게임 한 판 **60초 이내**, 결과·정산·"다시" 1탭 재시작이 되는가.
3. **캐시아웃 게임**(크래시·마인즈·타워·치킨로드·플립·하이로우): 큰 초록 버튼으로 중간 회수가 되는가. 버스트 시 꽝 처리되는가.
4. **리스크 슬라이더**(다이스·림보·휠·플린코 등) 움직이면 확률·배당 표시가 즉시 갱신되는가.
5. 큰 배수 승리 시 **빅윈/잭팟 연출**, 잭팟 후 거리 **전광판** 문구가 흐르는가.
6. 홈 하단 **내 전적**이 플레이에 따라 갱신되는가. 새로고침해도 유지되는가.
7. 성능: 게임·거리가 **끊김 없이(30fps)** 도는가. 거리가 길어졌으니 **자전거 타면 빨라지는가**.
